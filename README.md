## Como Rodar o Projeto?

Primeiro, instale as dependências e inicie o servidor de desenvolvimento:

```bash
npm i
# ou
npm install

yarn i
# ou
yarn install

npm run dev
# ou
yarn dev

Depois, abra http://localhost:3000 no seu navegador para ver o resultado.
